# CineReel

Next.js and MongoDB-powered movie database web app with shadcn for seamless browsing, adding favorites, writing reviews, accessing movie details and giving ratings.

## Features
- Find Movies
- Favorite Movies
- Review Movies
- Watch Movies Trailers
- View Movie Details

## Core Stack
- Next.js
- MongoDB
- Typescript
- shadcn
